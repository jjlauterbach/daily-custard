What is Daily Flavors?

Daily Flavors is a free service that aggregates the "Flavor of the Day" information from popular frozen custard shops in one convenient place. Instead of visiting multiple websites to find out what's being served today, you can check Daily Flavors and see everything at a glance.

Supported Locations

We currently track daily flavors from these frozen custard establishments:

  • Culver's - Multiple locations in the Milwaukee area
  • Kopp's Frozen Custard - A Wisconsin institution since 1950
  • Oscar's Frozen Custard - Local favorite
  • Murf's Frozen Custard - Family-owned classic
  • Bubba's Frozen Custard - Community favorite

We're always looking to add more locations! If you'd like to suggest a shop, please reach out.

How It Works

Our system automatically checks each shop's website daily to gather the current flavor information. This data is then compiled and displayed in an easy-to-read format. The information is refreshed each morning to ensure you always have the latest flavors.

Data Accuracy

While we strive to provide accurate information, please note that:

  • Flavors may occasionally change after our daily update
  • Some shops may run out of certain flavors during the day
  • We recommend calling ahead for special occasions

All flavor information is sourced directly from the official websites of each establishment. We are not affiliated with any of these businesses.

Open Source

Daily Flavors is an open-source project. You can find the source code, report issues, or contribute on GitHub.
